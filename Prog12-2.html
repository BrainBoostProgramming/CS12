Assignment 2

For this we're just adding some useful array-related functions to the class ArrayOps inside the ArrayDemo.java file in the Assignment 2 folder. The names of the functions are all in there already, you just need to write the actual code.

SumArray should return an int equal to the sum of the numbers stored in the array.

PrintBackwards should print out the items in the array in reverse order, from highest to lowest.

PrintOddElements should print out the values stored at odd numbered indices. (Note a better name for this function might be something like PrintOddIndexValues or something).

PrintEvenElements should do the same as above but with even numbered indices.

ZeroOut should set the value of each item in the array to 0.
