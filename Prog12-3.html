Assignment 3

Read this.

  1. Modify the PhoneBook.java program in the Assignment 3 folder so that it prompts for the name of the person to search for, as opposed to just hard-coding in "Violet Smith".

  2. Modify the program further so that it stores:

    • Name
    • Phone
    • Address
    • City
    • Province
    • Postal Code

  3. Add additional search functions to the PhoneBook class for each of these new fields.

  4. The user will choose which field they are searching for by inputting a number from 1 to 6. 1 for Name, 2 for Phone, and so on. Use a switch statement.

  5. Loop the program so that a user can do multiple searches. Include a way to quit the program (entering 0 for example).
