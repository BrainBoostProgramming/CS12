Assignment 8

This assignment will deal with text input in JavaFX.

For the HTML/JS part: The HTML file is like the FXML file in Java. All it does is define a layout/hierarchy. Actual logic goes into the JavaScript file. In the <head> tag of the HTML file we load the JS like so:

<script src="repeater.js"></script>

The browser then runs the code in the JS file.

We can assign ID's to HTML elements:

<input id="foo"></input>

Then we can get and change that element in JS using the ID:

let foo = document.getElementById('foo');

We can make use of various events using listeners:

// Tells the browser to run the function 'ready' once the HTML is fully loaded.
document.addEventListener('DOMContentLoaded', ready);

function fooEventHandler(event) {
    console.log('the foo element has received a new input event');
}

function ready() {
    let foo = document.getElementById('foo');

    // When the 'foo' element receives a new input, call the 'fooEventHandler' function.
    foo.addEventListener('input', fooEventHandler);
}
